# printingmagic.lens Governance

This is the LENS Workbench governance repository for the `printingmagic.lens` control repo.

## Contents

- `repo-inventory.yaml` — Registry of target project repositories managed by this control repo
- `feature-index.yaml` — Registry of all features (created on first `/new-feature`)
- `users/` — User profiles
- `features/` — Feature planning documents
